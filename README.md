# video-call

Experimental WebRTC videoconferencing app

## Client

### Environment

.env file

```
VITE_API_URI=[your_api_uri]
```

#### Setup

Make sure to install the dependencies:

```bash
# pnpm
pnpm install
```

#### Development Server

```bash
pnpm run dev
```

#### Production

Build the application for production:

```bash
pnpm run build
```

## API

### Environment

.env file

```
PORT=[your_api_port]
CLIENT_URI=[your_client_uri]
```

#### Setup

Make sure to install the dependencies:

```bash
# pnpm
```
